# TikZ/SVG 图示验证集

本验证集用于检查 `course-alchemist` 当前 TikZ 图生成流程是否能把具体图片需求转成可编译、可导出、可人工核对的静态 SVG。每张图都应先满足教学目标，再满足视觉要求；若二者冲突，以教学关系正确为先。

生成流程：

1. 先按本文件的图示需求构建显式 IR，保存为 `source/figure-ir.json`。IR 必须说明 `purpose`、`entities`、`required_semantics`，并尽量写出 `forbidden_semantics`。
2. 按 IR 生成 `figures/tikz/<figure-id>.tex`。TikZ 片段应只承载图形本身，不写 `figure`、`\caption` 或文档导言区。
3. 运行 `python scripts/validate_tikz_figures.py assets/examples/tikz-figure-validation --output .test_tmp/tikz-example-build`。验证脚本会自动读取 `source/figure-ir.json`，同时执行静态检查、IR 语义检查、编译和 SVG 导出。
4. 检查 `.test_tmp/tikz-example-build/report.md` 与 `manifest.json`，确认 `failed_count=0`、`semantic_error_count=0`。
5. 人工逐张检查 `.test_tmp/tikz-example-build/svg/*.svg`，确认图像关系、方向、标签、遮挡和裁切。源码级语义检查通过不等于视觉检查通过。

## 微积分

### calc-01-derivative-tangent

- 教学目标：说明导数来自割线斜率趋近切线斜率。
- 图像内容：坐标轴、一条上凸曲线、曲线上两点 `P` 与 `Q`、过 `P` 的切线、过 `P,Q` 的割线、水平小段 `\Delta x`。
- 必须满足：`P` 和 `Q` 都在曲线上；切线只在 `P` 附近贴近曲线；割线必须穿过两点；`\Delta x` 位于两点的水平投影之间。
- 不应出现：切线与割线混淆、标签压住曲线或点、曲线被裁切。
- 建议图注：割线斜率随 `Q` 靠近 `P` 而趋近于 `P` 处切线斜率。

### calc-02-riemann-area

- 教学目标：说明定积分近似为曲线下方小矩形面积之和。
- 图像内容：坐标轴、区间 `[a,b]`、正函数曲线、5 个竖直矩形、曲线下方面域淡色填充。
- 必须满足：矩形底边完全位于 `[a,b]`；矩形上边大致贴近曲线；填充区域不越过坐标轴；`a`、`b` 标签在 x 轴上。
- 不应出现：矩形遮住整条曲线、矩形超出积分区间、曲线与坐标轴标签重叠。
- 建议图注：把 `[a,b]` 划分成小区间后，矩形面积和逼近曲线下方面积。

### calc-03-polar-sector

- 教学目标：说明极坐标二重积分中的扇环区域及面积元方向。
- 图像内容：扇环区域 `D`，内外两条圆弧 `r=r_1` 与 `r=r_2`，两条射线边界 `\theta=\alpha` 与 `\theta=\beta`，一个小面积元，径向箭头 `r` 和角向箭头 `\theta`。
- 必须满足：区域是扇环而不是整圆；小面积元位于区域内部；角向箭头沿逆时针；标签不遮挡边界。
- 不应出现：内外半径标反、角向箭头顺时针、面积元跑到区域外。
- 建议图注：扇环区域中面积元可写作 `r\,dr\,d\theta`。

## 普通物理

### phys-01-projectile-motion

- 教学目标：说明斜抛运动中速度分解与重力方向。
- 图像内容：抛物线轨迹、初速度 `v_0`、水平分量 `v_{0x}`、竖直分量 `v_{0y}`、重力加速度 `g` 向下、地面线。
- 必须满足：轨迹先上升后下降；`v_{0x}` 向右；`v_{0y}` 向上；`g` 全程向下。
- 不应出现：重力方向向上、初速度分量方向反、轨迹与地面标签重叠。
- 建议图注：斜抛运动可分解为水平方向匀速运动和竖直方向匀变速运动。

### phys-02-point-charge-field

- 教学目标：说明正点电荷电场线沿径向向外。
- 图像内容：中心正点电荷 `+q`，8 条径向电场线，箭头全部向外，可加一条虚线等势圆作为辅助。
- 必须满足：电场线从中心向外发散；任意两条电场线不相交；箭头方向全部远离电荷；等势圆不能被误认为电场线。
- 不应出现：箭头向内、场线相交、`+q` 标签遮住箭头。
- 建议图注：正点电荷的电场方向沿径向向外。

### phys-03-inclined-plane-forces

- 教学目标：说明斜面上物块的受力分解。
- 图像内容：倾角为 `\theta` 的斜面、物块、重力 `mg` 竖直向下、支持力 `N` 垂直斜面、摩擦力 `f` 沿斜面向上、重力沿斜面和垂直斜面的虚线分量。
- 必须满足：`mg` 竖直向下；`N` 垂直斜面；`f` 沿斜面向上；分量箭头与斜面方向一致。
- 不应出现：支持力竖直向上、摩擦力脱离斜面、角度 `\theta` 标在错误位置。
- 建议图注：重力可分解为沿斜面与垂直斜面的两个分量。

## 线性代数

### linalg-01-linear-transform-grid

- 教学目标：说明线性变换把基向量和网格同时映到新方向。
- 图像内容：左侧原坐标网格和基向量 `e_1,e_2`，右侧变换后平行四边形网格和向量 `Ae_1,Ae_2`，中间有箭头 `A`。
- 必须满足：右侧网格仍由两组平行线组成；`Ae_1` 与 `Ae_2` 不共线；变换箭头从左指向右。
- 不应出现：右侧网格变成随意曲线、两个像向量重合、左右图标签混乱。
- 建议图注：线性变换由基向量的像决定，并把方格映为平行四边形网格。

### linalg-02-projection-subspace

- 教学目标：说明向量到一维子空间的正交投影。
- 图像内容：一条通过原点的子空间直线 `L`，向量 `v`，投影 `proj_L v`，从 `v` 端点到投影点的虚线垂线，垂足直角标记。
- 必须满足：投影向量落在 `L` 上；残差向量垂直于 `L`；直角标记在垂足处。
- 不应出现：投影点偏离子空间、残差不垂直、标签遮挡直角标记。
- 建议图注：正交投影把向量分解为子空间方向和垂直方向两部分。

### linalg-03-eigenvector-action

- 教学目标：说明特征向量在线性变换下方向不变，只改变长度和可能改变方向。
- 图像内容：左侧向量 `v`，右侧同一直线方向上的 `Av=\lambda v`，另有一个非特征向量 `w` 与其像 `Aw` 改变方向，用淡色辅助线表示特征直线。
- 必须满足：`v` 与 `Av` 共线；`w` 与 `Aw` 不共线；变换箭头从左到右；特征直线穿过原点。
- 不应出现：`v` 与 `Av` 不共线、`w` 与 `Aw` 也共线、标签遮挡向量箭头。
- 建议图注：特征向量经过线性变换后仍在原来的直线上。
